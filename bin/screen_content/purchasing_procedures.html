Low Inventory Notice:

  1. Verify low or no inventory physically.
  2. Open Item List & locate part by Item # or item name.
  3. Verify no orders have already been placed on "Orders"

Reference Previous Vendor:

  1. Within Items -> History you will see Receive Items history.
  2. Double Click on Receive Items History and you will find a vendor name and yellow tab with a reference # or Order #, use this # to place another order from same vendor.
  3. If this item cannot be reordered from same vendor contact Store Manager for assistance.

Create New Purchase Order:

  1. Reference Vendor.
  2. Apply to shipping charges within Item Cost.
  3. Include Reference # within Instructions Section on New PO.
